# Tableau MCP Setup

## Prerequisites

* Tableau Cloud Account
* Claude Desktop
* Node.js

## Steps

1. Create a Personal Access Token (PAT) in Tableau Cloud
2. Copy `claude_desktop_config.example.json`
3. Replace placeholder values with your own credentials
4. Restart Claude Desktop
5. Verify connection by asking:

Show me all views in the workbook

## Security Note

Never commit real PAT tokens or Claude configuration files to GitHub.
